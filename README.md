# Kuifje

## Installation

Install the [Kuifje](https://github.com/martonbognar/kuifje) first


## Usage

on the directory on Kuifje_compiler/Kuifje/, run ghci and load the Run.hs file. 
```
ghci Run.hs
```

inside the GHCI

Run the file
```
*Run> runFile "../tests/<example file>"
```

Parse file only
```
*Run> parseFile "../tests/<example file>"
```
